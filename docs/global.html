delegate(el, eventType, targetSelector, handler, context)

Registers a handler on a delegate element which will handle event on behalf of delegator elements. The delegator element does not have to be the physical element on which the event originally fires on. Since the event bubbles up we can specify an ancestor element, defined by the targetSelector, on which we will call a handler along with a synthetic event as if the event fired on this element.
Parameters:
Name Type Description
el Element the delegate which will handle events on behalf of other elements
